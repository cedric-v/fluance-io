# Book a Fluance class

This page lets agents and users inspect available Fluance classes and start a booking.

## Flow

1. Inspect available classes with date, time, location, and remaining spots.
2. Select a class.
3. Provide an email address.
4. Use an active pass, claim the free first session, or pay per class / with a pass.

## Useful API

- `GET https://fluance.io/api/courses`
- `GET https://fluance.io/api/pass-status?email=...`
- `POST https://fluance.io/api/bookings`
