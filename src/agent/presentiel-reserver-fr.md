# Réserver un cours Fluance

Cette page permet de consulter les cours Fluance disponibles et de réserver une séance.

## Parcours

1. Consulter les cours disponibles avec date, heure, lieu et places restantes.
2. Choisir un cours.
3. Indiquer son email.
4. Utiliser un pass actif, réserver une séance d'essai offerte, ou payer à l'unité / avec un pass.

## API utile

- `GET https://fluance.io/api/courses`
- `GET https://fluance.io/api/pass-status?email=...`
- `POST https://fluance.io/api/bookings`
